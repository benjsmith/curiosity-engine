CLIP = Contrastive Language-Image Pre-training (vault:20260416-234959-local-clip-radford-2021.md.extracted.md). Radford et al. 2021 @ OpenAI, arXiv 2103.00020 (vault:20260416-234959-local-clip-radford-2021.md.extracted.md). Dual-encoder vision-language model (vault:20260416-234959-local-clip-radford-2021.md.extracted.md).

Two towers: image encoder ([[vit]] or [[resnet]] variants, vault:20260416-234959-local-resnet-he-2015.md.extracted.md) + text encoder ([[transformer]]) (vault:20260416-234959-local-clip-radford-2021.md.extracted.md). [[resnet]] = He 2015 residual-learning, F(x)+x shortcut blocks, bottleneck 1x1-3x3-1x1, up to 152 layers, ILSVRC 2015 winner (vault:20260416-234959-local-resnet-he-2015.md.extracted.md). CLIP tweak: antialiased blurpool + attention pool replaces global avg pool (vault:20260416-234959-local-clip-radford-2021.md.extracted.md). [[transformer]] = Vaswani 2017, stacked multi-head self-attention + position-wise FFN, scaled dot-product softmax(QK^T/sqrt(d_k))V, residuals + layernorm, sinusoidal positions (vault:20260416-234959-local-attention-is-all-you-need-vaswani-2017.md.extracted.md). CLIP text tower borrows Vaswani encoder recipe (vault:20260416-234959-local-attention-is-all-you-need-vaswani-2017.md.extracted.md). Project both to shared embedding space, dim 512-1024 (vault:20260416-234959-local-clip-radford-2021.md.extracted.md). Contrastive InfoNCE loss: pull matched (image, caption) pairs close, push N^2-N mismatched apart (vault:20260416-234959-local-clip-radford-2021.md.extracted.md). Embeddings L2-normalized, logits scaled by learned temperature tau (vault:20260416-234959-local-clip-radford-2021.md.extracted.md). Symmetric cross-entropy over image-to-text + text-to-image matrices (vault:20260416-234959-local-clip-radford-2021.md.extracted.md).

Dataset: WIT (WebImageText), 400M (image, text) pairs scraped public internet (vault:20260416-234959-local-clip-radford-2021.md.extracted.md). 500,000 queries from English Wikipedia words, bi-grams, WordNet synsets, capped 20,000 per query (vault:20260416-234959-local-clip-radford-2021.md.extracted.md). Word count comparable WebText/GPT-2 (vault:20260416-234959-local-clip-radford-2021.md.extracted.md). No manual labels (vault:20260416-234959-local-clip-radford-2021.md.extracted.md). Extends distributional semantics lineage ([[word2vec]], vault:20260416-234959-local-word2vec-mikolov-2013.md.extracted.md) to multimodal.

Pretraining objective contrast vs [[bert]]: [[bert]] does masked-LM (MLM), 15% WordPiece tokens masked, predict originals, bidirectional encoder (vault:20260416-234959-local-bert-devlin-2018.md.extracted.md). CLIP skips token reconstruction; uses contrastive alignment between modalities (vault:20260416-234959-local-clip-radford-2021.md.extracted.md). Contrastive 3-4x more sample-efficient than generative caption prediction, 3x better than bag-of-words (vault:20260416-234959-local-clip-radford-2021.md.extracted.md). Text encoder: 63M-param, 12-layer, 512-wide Transformer, 8 heads (Vaswani multi-head spec, vault:20260416-234959-local-attention-is-all-you-need-vaswani-2017.md.extracted.md), BPE vocab 49,152, max 76 tokens, [EOS] activation projected (vault:20260416-234959-local-clip-radford-2021.md.extracted.md). Image encoder family: ResNet-50/101/50x4/50x16/50x64 (bottleneck residual backbones, vault:20260416-234959-local-resnet-he-2015.md.extracted.md) with antialiased blurpool + attention pooling, or ViT-B/32, ViT-B/16, ViT-L/14 (vault:20260416-234959-local-clip-radford-2021.md.extracted.md). Trained 32 epochs, batch 32,768, Adam decoupled weight decay, cosine decay, 10k warmup, mixed precision, all-gather across GPUs (vault:20260416-234959-local-clip-radford-2021.md.extracted.md). tau learned in log space, clipped logits < 100 (vault:20260416-234959-local-clip-radford-2021.md.extracted.md). ResNet-50x64: 12 days on 592 V100s; ViT-L/14: 18 days on 256 V100s (vault:20260416-234959-local-clip-radford-2021.md.extracted.md).

Zero-shot transfer: prompt "a photo of a {class}" -> classify via cosine similarity vs image embed (vault:20260416-234959-local-clip-radford-2021.md.extracted.md). Prompt ensembling over 80 prompts adds ~5 points (vault:20260416-234959-local-clip-radford-2021.md.extracted.md). ViT-L/14 @ 336px: 76.2% zero-shot ImageNet top-1, matches supervised ResNet-50 76.1% (vault:20260416-234959-local-clip-radford-2021.md.extracted.md). Evaluated 30+ datasets: OCR, action recognition, geo-localization, fine-grained classification (vault:20260416-234959-local-clip-radford-2021.md.extracted.md). Beats ResNet-50 linear probe on 16 of 27 datasets (vault:20260416-234959-local-clip-radford-2021.md.extracted.md). Linear-probe ViT-L/14 SOTA on 21 of 27, avg 84.5% vs 73.5% supervised ResNet-50 (vault:20260416-234959-local-clip-radford-2021.md.extracted.md). Underperforms EuroSAT, PatchCamelyon, CLEVR counting (vault:20260416-234959-local-clip-radford-2021.md.extracted.md).

Robust distribution shift: ImageNet-V2 70.1% vs ResNet-101 64.3%; ImageNet-Sketch 60.2% vs 24.1%; ImageNet-A 77.2% vs 2.7%; ImageNet-R 88.9% vs 37.7% (vault:20260416-234959-local-clip-radford-2021.md.extracted.md). "Effective robustness" concept introduced (vault:20260416-234959-local-clip-radford-2021.md.extracted.md). Emergent OCR: SST-2 rendered as images 67.9%; Country211 geo 31.3% (vault:20260416-234959-local-clip-radford-2021.md.extracted.md).

Lineage: VirTex, ConVIRT, ICMLM precursors; key finding scale + simple contrastive surpasses supervised transfer (vault:20260416-234959-local-clip-radford-2021.md.extracted.md). Unlike [[bert]] fine-tune paradigm (add output head, update all params, vault:20260416-234959-local-bert-devlin-2018.md.extracted.md), CLIP ships zero-shot via prompt embeddings (vault:20260416-234959-local-clip-radford-2021.md.extracted.md). Seeded [[dalle-2]], Stable Diffusion text conditioning, open-clip, LiT, [[flamingo]] (vault:20260416-234959-local-clip-radford-2021.md.extracted.md). Foundation for modern [[multimodal]] retrieval + generation (vault:20260416-234959-local-clip-radford-2021.md.extracted.md). Contrast captioning [[blip]]. Limitations: fine-grained/abstract tasks, prompt sensitivity, web-scrape bias (vault:20260416-234959-local-clip-radford-2021.md.extracted.md).
