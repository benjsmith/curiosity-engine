T5 = Text-to-Text Transfer Transformer. Raffel et al. 2019 @ Google (vault:20260416-234959-local-t5-raffel-2019.md.extracted.md). Encoder-decoder [[transformer]]. Sizes: small 60M -> 11B params.

Unifying frame: every NLP task = text-in / text-out. Translate, classify, summarize, QA all cast as seq2seq. Task prefix tokens signal target ("translate English to German: ...").

Pre-train on C4 (Colossal Clean Crawled Corpus), 750GB filtered web. Span-corruption objective: mask contiguous spans, predict concatenated sequence. Generalizes [[bert]] MLM (vault:20260416-234959-local-bert-devlin-2018.md.extracted.md) to generative form.

Relative position biases (T5-style), shared across layers. [[adam]]-family optimizer Adafactor for memory (vault:20260416-234959-local-adam-kingma-2014.md.extracted.md). SentencePiece 32K vocab.

Systematic ablation study: architecture, objective, corpus, training strategy. SOTA GLUE, SuperGLUE, SQuAD, CNN/DM. 11B model strongest. Influenced [[flan-t5]] instruction tuning, [[byte-t5]], [[ul2]]. Standard seq2seq baseline. Contrast decoder-only [[gpt3]], [[llama]].
