State-of-the-art computer vision systems at the time of the paper were trained on fixed, crowd-labeled datasets such as ImageNet with a predefined set of object categories. This paradigm limits models in two ways: (i) acquiring additional labeled data is expensive, and (ii) the resulting classifier is only able to predict the classes it was trained on, so zero-shot transfer to new categories typically requires fine-tuning on new labeled data.
