# Welcome

This wiki is a [[curiosity-engine]] knowledge base — a self-improving notebook curated on top of a [[vault]] of raw sources. Pages are typed by role; the curator uses the `[[wikilink]]` and `(vault:path)` conventions to keep claims traceable.

## Browse by page type

- [[entities]] — named things (people, organisations, models, systems, places).
- [[concepts]] — abstract ideas, methods, phenomena.
- [[evidence]] — single-source observations (one claim, one citation).
- [[facts]] — atomic numerical or factual anchors.
- [[analyses]] — multi-source syntheses with open questions.
- [[sources]] — index stubs for every vault source.
- [[tables]] — comparison / cross-section tables.
- [[figures]] — extracted or authored figures with captions.
- [[notes]] — personal note surface; drains into topic files.
- [[todos]] — to-do buckets (day / month / year priority).

## How to use

- **Query:** ask the curator ("what do I know about X?") — it reads the relevant pages and answers with citations.
- **Add a source:** drop a file in `vault/raw/` and say "ingest this", or use `/note` / `/todo` for your own thoughts.
- **Curate:** say "curate this wiki for an hour" to run the autonomous improvement loop.

The index is kept intentionally short — `.curator/index.md` holds the full page catalog (auto-regenerated by `sweep.py fix-index`).
